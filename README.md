Vue CLI3<br/>
Vue components<br/>
Vue router<br/>
Vue Vuex<br/>
Vue 套件 - veeValidate，vue-loading-overlay，vue-axios<br/>
Bootstrap 4並進行客製化<br/>
手刻Sass/Pug

後台: 產品管理，訂單管理，優惠券管理
